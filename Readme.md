
# Result Fetcher

This is a simple web scrapper used for retrieval and analysis of result from NIT(National Institute of Technology) Raipur website. This project uses requests, pdfplumber, io, urllib3, Beautiful Soup and Tkinter to achieve this.
## Features
1. Retrieves data from NIT website
2. Retrieves data of indivisual as well as branch wise.
3. Uses tkinter to show data


## Prerequisites
Before running the project on your system, make sure you have the following libraries installed:
1. Requests
2. io
3. PdfPlumber
4. Beautiful Soup
5. Regular Expression

## Usage
1. Clone the repositry to your local system.
2. Make sure pip is installed on your system.
3. Create a virtual environment on your system.
4. Install the packages from requirements.txt.
5. Run the `program_gui.py` script.

## How it works
1. It retrieves data from NIT Raipur website.
2. It can be branch wise or indivisual.
3. It has a sorting feature by spi or cpi.
